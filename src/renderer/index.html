    Help

    Folder & Playback

    • Select Folder — Choose a folder of images for the player.
    • Start — Begin playback of the selected folder.
    • Stop — Pause playback.

    Transitions & Timing

    • Transition — Choose Cut (instant) or Crossfade.
    • Fade — Crossfade duration (0.5-5s).
    • Display Time — Time each image stays on screen.
    • Global Transition Footer — Toggle with Cmd+G and optionally override transition settings for all players.

    Image Layout

    • Scale to Fill — Crop to fill the frame.
    • Color — Set a background color.

    Output

    • Output — Choose a display.

    Schedules

    • Add Schedule — Create as many timed folder switches as you need for each player.
    • Enable schedule — Activates the selected schedule slot.
    • Days — Choose which weekdays each scheduled folder switch can run on.

    Player Titles

    • Double-click player name — Edit the title inline.

    Keyboard Shortcuts

    • Cmd+1 — 2 Player layout.
    • Cmd+2 — 4 Player layout.
    • Cmd+3 — 2 x 2 layout.
    • Cmd+4 — 1 x 4 layout.
    • Cmd+D — Debug.
    • Cmd+E — Export player configurations.
    • Cmd+F — Always on Top.
    • Cmd+G — Global Transition.
    • Cmd+H — Help.
    • Cmd+O — Output.
    • Cmd+P — Stop All Players.
    • Cmd+S — Schedule.
    • Cmd+T — Transition.
